# CS 300 — Data Structures & Algorithms (C++)

Implemented and compared core data structures, analyzing runtime trade-offs.

## ABCU Course Advising Program
Loads a CSV course catalog, validates each record (including verifying every listed
prerequisite exists), stores courses, and prints an alphanumerically ordered list.
Built and evaluated hash table, vector, and BST versions to recommend the best fit.
- `ABCU_HashTable_Pseudocode.docx`
- `Code_Reflection.docx`

## Data structure implementations
- `BinarySearchTree.cpp` — insert, search, in-order traversal
- `LinkedList.cpp` — singly linked list (append, prepend, search, remove, display)
- `VectorSorting.cpp` — selection sort & quicksort with partition logic

**Skills:** hash tables, binary search trees, linked lists, sorting algorithms,
pointers & dynamic memory, Big-O analysis
