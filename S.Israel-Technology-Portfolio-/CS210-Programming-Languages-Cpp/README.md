# CS 210 — Programming Languages (C++)

Applied C++ projects focused on clean class design, file handling, and a usable interface.

## Corner Grocer — Item Tracking Program
Reads a grocery transaction file, normalizes input, and counts item frequency with `std::map`.
Lets the user look up any item, prints a full frequency report and histogram, and writes a backup data file.
- `CornerGrocer.cpp` — source
- `CornerGrocer_input.txt` — sample input

**Skills:** C++ classes, `std::map`, file I/O, input validation, menu-driven UI

## Airgead Banking — Investment Calculator
Object-oriented compound-interest calculator that projects investment growth year by year,
with and without recurring monthly deposits, and prints formatted year-end reports.
- `InvestmentCalculator.cpp` / `InvestmentCalculator.h`
- `Airgead_Banking_Pseudocode.docx` — design

**Skills:** OOP, compound-interest logic, formatted output
