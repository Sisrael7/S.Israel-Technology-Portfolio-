# MAT 350 — Applied Linear Algebra

## Network Capacity & Data-Rate Analysis
Modeled a communications network as a system of links and capacities. Used linear
algebra (and MATLAB) to calculate the data rate on each link, compare it to recommended
capacity, and recommend which links to keep vs. upgrade — turning matrix math into
infrastructure decisions.
